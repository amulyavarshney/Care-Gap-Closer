A care-coordination assistant that reviews a patient's FHIR record for overdue preventive care — screenings (diabetes, cancer, bone density, depression), monitoring (blood pressure, lipids), and vaccines (flu, Tdap, pneumococcal, shingles) — against USPSTF and ACIP guidelines, then drafts ready-to-send SMS or patient-portal outreach to close each gap. Every gap is grounded in structured evidence from the chart, never guessed. Requires a connected FHIR data source (SMART-on-FHIR) with patient context.
